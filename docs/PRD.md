## 📌 产品概述

### 产品定位
Commit 是一款将 Git 版本控制理念（仓库、分支、提交）应用于个人任务管理的效率工具。通过创新的分支管理和可视化 Git Graph，让用户像管理代码一样管理待办事项。

### 目标用户
- 个人开发者
- 项目经理
- 学生
- 任何需要高效任务管理的个人用户

### 核心价值
- **免费无广告**：完全免费，无会员系统
- **离线优先**：所有功能本地可用，不依赖网络
- **跨平台**：一套代码覆盖 4 个平台
- **数据安全**：本地存储，隐私优先

## 📋 详细功能说明

### 模块 1：仓库管理 (P0)

#### 功能点

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| R1 | 创建仓库 | 输入名称、选择图标和颜色 | P0 |
| R2 | 编辑仓库 | 修改名称、图标、颜色 | P0 |
| R3 | 删除仓库 | 软删除，可恢复 | P0 |
| R4 | 归档仓库 | 归档后不显示在主页 | P1 |
| R5 | 仓库列表 | 显示所有仓库，支持排序 | P0 |
| R6 | 仓库搜索 | 按名称搜索仓库 | P1 |

#### 业务规则
- 每个仓库自动创建一个 `main` 主分支
- 仓库删除后，关联的分支和任务一起软删除
- 仓库图标支持 Emoji 选择器
- 仓库颜色支持预设色板

---

### 模块 2：分支管理 (P0)

#### 功能点

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| B1 | 创建分支 | 从当前分支创建新分支 | P0 |
| B2 | 切换分支 | 切换到其他分支查看任务 | P0 |
| B3 | 合并分支 | 将分支任务合并到主分支 | P0 |
| B4 | 删除分支 | 删除分支及其任务 | P0 |
| B5 | 重命名分支 | 修改分支名称 | P1 |
| B6 | 分支列表 | 显示仓库下所有分支 | P0 |

#### 业务规则
- `main` 分支不可删除
- 合并分支时，未完成的任务会移动到目标分支
- 分支支持颜色标记
- 分支支持折叠/展开

### 模块 3：任务管理 (P0)

#### 功能点

| 编号 | 功能 | 说明 | 优先级 |
|------|------|------|--------|
| T1 | 创建任务 | 输入标题、描述、截止日期 | P0 |
| T2 | 编辑任务 | 修改任务所有属性 | P0 |
| T3 | 完成任务 | 标记任务为已完成 | P0 |
| T4 | 删除任务 | 软删除任务 | P0 |
| T5 | 子任务 | 创建层级任务关系 | P1 |
| T6 | 任务排序 | 按优先级/日期/创建时间排序 | P1 |
| T7 | 任务筛选 | 按状态/优先级筛选 | P1 |
| T8 | 截止日期 | 设置截止日期和提醒 | P0 |
| T9 | 任务优先级 | 设置高/中/低优先级 | P0 |

#### 任务状态
```
待办 (Todo) → 进行中 (InProgress) → 已完成 (Done)
                ↓
            已取消 (Cancelled)
```

#### 优先级
- 🔴 高优先级 (High)
- 🟡 中优先级 (Medium)
- 🟢 低优先级 (Low)

## 🎨 UI 规范

### 设计原则
1. **简约**：界面清爽，减少视觉干扰
2. **一致**：跨平台保持统一的设计语言
3. **高效**：操作路径最短，快速完成任务
4. **优雅**：遵循优雅设计风格（elegant skill）

### 配色方案
- **主色调**：待定（建议使用单一强调色）
- **背景色**：系统默认（跟随深色/浅色模式）
- **文字色**：系统默认

### 字体
- **iOS/Mac**：SF Pro
- **Android**：Roboto
- **Windows**：Segoe UI

## 📊 数据模型

### 核心实体

#### Repository (仓库)
```json
{
  "id": "UUID",
  "name": "String",
  "icon": "String (Emoji)",
  "color": "String (Hex)",
  "isArchived": "Boolean",
  "createdAt": "DateTime",
  "updatedAt": "DateTime"
}
```

#### Branch (分支)
```json
{
  "id": "UUID",
  "repositoryId": "UUID",
  "name": "String",
  "parentBranchId": "UUID (nullable)",
  "isMain": "Boolean",
  "color": "String (Hex)",
  "createdAt": "DateTime"
}
```

#### Task (任务)
```json
{
  "id": "UUID",
  "branchId": "UUID",
  "title": "String",
  "description": "String (nullable)",
  "status": "Enum (Todo/InProgress/Done/Cancelled)",
  "priority": "Enum (High/Medium/Low)",
  "dueDate": "DateTime (nullable)",
  "completedAt": "DateTime (nullable)",
  "parentTaskId": "UUID (nullable)",
  "createdAt": "DateTime",
  "updatedAt": "DateTime"
}
```

#### Commit (提交历史)
```json
{
  "id": "UUID",
  "taskId": "UUID",
  "branchId": "UUID",
  "message": "String",
  "type": "Enum (Create/Update/Merge/Complete)",
  "createdAt": "DateTime"
}
```
